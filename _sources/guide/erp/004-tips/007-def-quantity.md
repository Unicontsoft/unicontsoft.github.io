# **Количество по подразбиране**

При създаване на нов документ (покупка, продажба, складов документ и др.) системата автоматично обзавежда полетата в колона **Количество** със стойност по подразбиране. 

Първоначалната системна настройка за количество по подразбиране може да бъде променена от **Средства » Настройки** в лентата с основно меню.  
Системата отваря форма, в която се избира секция **Контейнер**. В полето за **Количество по подразбиране** се въвежда желаната стойност. Направените промени се потвърждават с бутон [**Ок**].
