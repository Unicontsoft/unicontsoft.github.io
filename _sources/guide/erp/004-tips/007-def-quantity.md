# **Количество по подразбиране**

При създаване на нов документ (покупка, продажба, складов документ и др.) системата обзавежда полетата в колона **Количество** с настроената стойност по подразбиране.  
Можете да промените тази първоначална системна настройка.  

От лентата с основно меню изберете **Средства » Настройки**.  
Навигирайте се до секция **Контейнер**. В полето за **Количество по подразбиране** въведете желаната стойност.  
Запишете промените с бутон [**Ок**].
