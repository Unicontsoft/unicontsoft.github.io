# **Разделяне на ред в продажба**

Системата разполага с инструмент за разделяне на вече въведен ред в документ за продажба. Когато за продукта има попълнено и общо количество, системата следи то да се запази при разделянето.  

Във форма за редакция на продажба маркирайте избрания ред. С десен клик изберете опция **Разделяне на ред**.  

![](916-line-item-split1.png){ class=align-center w=15cm }

С това отваряте формата за разделяне.  
В нея има добавен допълнителен ред с продукта. Попълнете желаната стойност в поле **Количество**. С това количество намалявате общото количество на оригиналния ред.  

![](916-line-item-split2.png){ class=align-center }

Потвърдете промяната с [**Enter**].  
Системата преизчислява количеството на оригиналния ред.  

![](916-line-item-split3.png){ class=align-center }
 
Потвърдете разделянето на реда с бутон [**OK**] и системата ще го приложи в продажбата.  
С [**Отказ**] можете да отмените направените модификации.  

![](916-line-item-split4.png){ class=align-center w=15cm }

Продължете обработка на документа без особености.
